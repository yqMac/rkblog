表示两种相互对立的状态间的切换，多用于触发「开/关」。
<br>
见[Switch 开关](https://element.eleme.cn/#/zh-CN/component/switch)


```vue
<el-switch
    v-model="showBid"
    style="display: block"
    active-color="#13ce66"
    inactive-color="#ff4949"
    active-text="bid"
    inactive-text="商户名称"
/>
```
